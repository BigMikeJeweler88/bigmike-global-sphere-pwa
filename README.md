# BigMike Global Sports CRM — PWA v2.0

Mobile-first PWA for the BigMike Global Sports Client Intelligence Platform.

## Sections
- Situation Room (JARVIS briefing, live ticker)
- Sports (1,572 athletes)
- Music (540 artists)
- Civilians (1,987 jewelry prospects)
- Influencers (140+ creators)
- Sales (transactions, payouts)
- System (health, Logic Auditor, QA Brain)
- Alerts, Birthdays, Highlights, Outreach, Search

## Setup
```bash
npm install
echo "NEXT_PUBLIC_REPLIT_URL=https://global-sphere--BigMike88.replit.app" > .env.local
npm run dev
```

## Deploy to Vercel
1. Connect this repo to Vercel
2. Set env var: `NEXT_PUBLIC_REPLIT_URL=https://global-sphere--BigMike88.replit.app`
3. Deploy
4. Open on iPhone Safari → Share → Add to Home Screen
